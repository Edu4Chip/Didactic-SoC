## Prerequisites
The required tools should be pre-installed on the virtual machine provided by the course. The specific required tools are:
- [Bender](https://github.com/pulp-platform/bender) - A dedicated dependency management tool for hardware projects.
- [Verilator](https://github.com/verilator/verilator) - A lightweight, open-source RTL simulator.
- [RISC-V GNU Toolchain](https://github.com/riscv-collab/riscv-gnu-toolchain) - RISC-V compiler toolchain.
